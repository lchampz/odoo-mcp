# Odoo CRM MCP Server

Full-featured MCP server for Odoo CRM. Manage leads, opportunities, pipeline stages,
sales teams, activities, contacts, tags, and analytics — all via AI.

## Configuration

Set these secrets in your Space settings:

| Secret | Description |
|---|---|
| `ODOO_URL` | Base URL (e.g. `https://mycompany.odoo.com`) |
| `ODOO_DATABASE` | Database name |
| `ODOO_USERNAME` | Login email |
| `ODOO_PASSWORD` | Password or API key |

## MCP URL

```
https://YOUR-USERNAME-odoo-crm-mcp.hf.space/mcp
```
